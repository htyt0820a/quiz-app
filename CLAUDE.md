# quiz-app

一般常識クイズアプリ。HTML/CSS/JavaScriptのみで構成されたフロントエンド完結型アプリ。

## 技術スタック

- **HTML** — マークアップ・画面構成
- **CSS** — スタイリング（フレームワークなし）
- **JavaScript** — ゲームロジック・DOM操作（バンドラー・フレームワークなし）

## ファイル構成

```
quiz-app/
├── index.html        # エントリーポイント
├── css/
│   └── style.css     # スタイル
├── js/
│   └── main.js       # アプリロジック
└── data/
    └── questions.js  # クイズ問題データ
```

## 開発ルール

- 外部ライブラリ・フレームワークは使用しない（バニラJS）
- `<script type="module">` を使い、ES Modules でモジュール分割する
- CSSはクラスベースで記述し、IDセレクタはJS用途に限定する
- クイズデータは `data/questions.js` に集約し、ロジックと分離する

## アプリの仕様

- 一般常識（歴史・地理・科学・文化など）の4択クイズ
- 出題はランダム順
- 正解・不正解のフィードバックを即時表示
- 全問終了後にスコアを表示し、リトライ可能

## 実行方法

ブラウザで `index.html` を直接開く、またはローカルサーバーで起動する。

```bash
# Python でローカルサーバーを起動する場合
python -m http.server 8080
```
